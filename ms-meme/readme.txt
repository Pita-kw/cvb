Description:
This resource lets you attach meme faces to players heads.
Hit F4 to open menu, select and apply color. The drawn face gets
bigger when players are in vehicle, also depthBias is set in order
to make the effect visible. You can add new faces yourself - 
just edit faces xml  and meta files and add .png images to faces directory.

have fun
